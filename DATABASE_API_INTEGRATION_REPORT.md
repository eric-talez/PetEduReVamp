## 🔄 실시간 데이터 흐름

### 데이터 흐름 검증
1. **Frontend (React)** → API 요청
2. **Express Routes** → 요청 처리
3. **Drizzle ORM** → 쿼리 실행
4. **PostgreSQL** → 데이터 저장/조회
5. **Response** → Frontend로 반환

### 성능 메트릭

- **활성 연결**: 5개
