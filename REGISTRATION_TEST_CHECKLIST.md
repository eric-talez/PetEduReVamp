# 등록 기능 전체 체크리스트

## 1. 회원가입 기능 (/auth/register)

### API 엔드포인트
- **POST** `/api/auth/register` (CSRF 보호 필요)
- **GET** `/api/auth/csrf` (CSRF 토큰 조회)

### 테스트 시나리오

#### 1.1 페이지 접근 및 UI 검증
- [ ] `/auth/register` 페이지 접근 가능
- [ ] 페이지 타이틀 "Talez 회원가입" 표시
- [ ] 소셜 로그인 버튼 표시 (카카오, 네이버)
- [ ] 모든 필수 입력 필드 표시:
  - [ ] 아이디
  - [ ] 이메일
  - [ ] 이름
  - [ ] 휴대폰 번호
  - [ ] 생년월일
  - [ ] 성별
  - [ ] 비밀번호
  - [ ] 비밀번호 확인
  - [ ] 사용자 유형
- [ ] "회원가입" 버튼 표시
- [ ] "로그인" 링크 표시

#### 1.2 필드 유효성 검증
- [ ] 빈 폼 제출 시 HTML5 유효성 검증 작동
- [ ] 비밀번호 불일치 시 에러 메시지 표시
- [ ] 휴대폰 번호 자동 포맷팅 (010-0000-0000)
- [ ] 생년월일 제한 (만 14세 이상)
- [ ] 이메일 형식 검증

#### 1.3 반려동물 보호자 회원가입
**테스트 데이터:**
```
아이디: testuser123
이메일: testuser@test.com
이름: 테스트사용자
휴대폰: 010-1234-5678
생년월일: 1990-01-01
성별: 남성
비밀번호: Test1234!@
사용자 유형: 반려동물 보호자
```

**확인사항:**
- [ ] 회원가입 성공 토스트 메시지 표시
- [ ] 자동 로그인 처리
- [ ] `/dashboard` 페이지로 리다이렉트

#### 1.4 훈련사 회원가입
**테스트 데이터:**
```
아이디: trainer123
이메일: trainer@test.com
이름: 테스트훈련사
휴대폰: 010-2345-6789
생년월일: 1985-05-15
성별: 여성
비밀번호: Trainer1234!@
사용자 유형: 훈련사
기관 코드: INST001
```

**확인사항:**
- [ ] 사용자 유형 "훈련사" 선택 시 기관 코드 필드 표시
- [ ] 회원가입 성공 처리
- [ ] 적절한 대시보드로 리다이렉트

#### 1.5 기관 관리자 회원가입
**테스트 데이터:**
```
아이디: admin123
이메일: admin@test.com
이름: 테스트관리자
휴대폰: 010-3456-7890
생년월일: 1980-03-20
성별: 남성
비밀번호: Admin1234!@
사용자 유형: 기관 관리자
기관 코드: INST001
```

**확인사항:**
- [ ] 사용자 유형 "기관 관리자" 선택 시 기관 코드 필드 표시
- [ ] 회원가입 성공 처리
- [ ] 적절한 대시보드로 리다이렉트

#### 1.6 에러 처리
- [ ] 중복 아이디 등록 시도 시 에러 메시지 표시

## 2. 훈련사 등록 기능 (/admin/trainers)

### API 엔드포인트
- **POST** `/api/trainers/register` (CSRF 보호 필요)

### 테스트 시나리오

#### 2.1 관리자 권한 확인
- [ ] 관리자 계정으로 로그인
- [ ] `/admin` 페이지 접근
- [ ] "훈련사 관리" 메뉴 클릭

#### 2.2 훈련사 추가 다이얼로그
- [ ] "새 훈련사 등록" 버튼 클릭
- [ ] 다이얼로그 팝업 표시
- [ ] 입력 필드 확인:
  - [ ] 이름
  - [ ] 이메일
  - [ ] 전화번호
  - [ ] 소속 기관
  - [ ] 자격증
  - [ ] 경력
  - [ ] 전문 분야

#### 2.3 훈련사 등록 테스트
**테스트 데이터:**
```
이름: 김훈련
이메일: trainer-new@example.com
전화번호: 010-9999-8888
소속 기관: (드롭다운에서 선택)
자격증: KKF 반려견 훈련사 1급
경력: 5년
전문 분야: 기본 훈련, 행동 교정
```

**확인사항:**
- [ ] 모든 필드 입력 후 "등록" 버튼 활성화
- [ ] 등록 성공 메시지 표시
- [ ] 훈련사 목록에 새 훈련사 추가됨
- [ ] 다이얼로그 자동 닫힘

#### 2.4 필수 필드 검증
- [ ] 필수 필드 누락 시 경고 메시지
- [ ] 이메일 형식 검증
- [ ] 전화번호 형식 검증

## 3. 반려동물 등록 기능 (/my-pets)

### API 엔드포인트
- **GET** `/api/pets` (반려동물 목록 조회)
- **POST** `/api/pets` (CSRF 보호, Zod 검증)
- **PUT** `/api/pets/:id` (반려동물 수정)
- **DELETE** `/api/pets/:id` (반려동물 삭제)

### 테스트 시나리오

#### 3.1 페이지 접근
- [ ] 사용자 계정으로 로그인 (반려동물 보호자)
- [ ] `/my-pets` 페이지 접근
- [ ] 반려동물 목록 표시 (비어있거나 기존 데이터)

#### 3.2 반려동물 추가
- [ ] "반려동물 추가" 버튼 클릭
- [ ] 다이얼로그 팝업 표시
- [ ] 입력 필드 확인:
  - [ ] 이름
  - [ ] 종 (개/고양이)
  - [ ] 품종
  - [ ] 나이
  - [ ] 성별
  - [ ] 몸무게
  - [ ] 털 색상
  - [ ] 성격
  - [ ] 병력
  - [ ] 특이사항
  - [ ] 이미지 업로드

#### 3.3 반려동물 등록 테스트
**테스트 데이터:**
```
이름: 뽀삐
종: 개 (dog)
품종: 골든 리트리버
나이: 3
성별: 남 (male)
몸무게: 25
털 색상: 황금색
성격: 활발하고 사람을 좋아함
병력: 없음
특이사항: 간식을 매우 좋아함
```

**확인사항:**
- [ ] 모든 필드 입력 가능
- [ ] 이미지 업로드 기능 작동 (선택사항)
- [ ] "저장" 버튼 클릭 시 성공 메시지
- [ ] 반려동물 목록에 새 반려동물 추가됨
- [ ] 반려동물 카드에 정보 정확히 표시

#### 3.4 반려동물 수정
- [ ] 등록된 반려동물 카드의 "수정" 버튼 클릭
- [ ] 기존 정보가 폼에 채워짐
- [ ] 정보 수정 (예: 나이 3 → 4)
- [ ] "저장" 버튼 클릭
- [ ] 수정 성공 메시지 표시
- [ ] 반려동물 목록에 수정된 정보 반영

#### 3.5 반려동물 삭제
- [ ] 반려동물 카드의 "삭제" 버튼 클릭
- [ ] 확인 다이얼로그 표시
- [ ] "확인" 클릭
- [ ] 삭제 성공 메시지 표시
- [ ] 반려동물 목록에서 제거됨

#### 3.6 이미지 업로드 테스트
- [ ] "이미지 업로드" 버튼 클릭
- [ ] 파일 선택 다이얼로그 표시
- [ ] 이미지 파일 선택
- [ ] 업로드 진행 표시
- [ ] 이미지 미리보기 표시
- [ ] 이미지 제거 기능 작동

## 4. 기관 등록 기능 (/institutes/register)

### 테스트 시나리오

#### 4.1 페이지 접근
- [ ] `/institutes/register` 페이지 접근
- [ ] 페이지 타이틀 "교육 기관 등록 요청" 표시
- [ ] 진행 상태 표시 (1단계 / 2단계)

#### 4.2 1단계: 기본 정보
**입력 필드:**
- [ ] 교육 기관명
- [ ] 교육 기관 유형 (드롭다운)
- [ ] 대표자명
- [ ] 사업자 등록번호
- [ ] 전화번호
- [ ] 이메일
- [ ] 웹사이트 (선택사항)
- [ ] 주소
- [ ] 상세 주소 (선택사항)

**테스트 데이터:**
```
교육 기관명: 테스트 훈련소
교육 기관 유형: 종합 교육
대표자명: 홍길동
사업자 등록번호: 123-45-67890
전화번호: 02-1234-5678
이메일: test-institute@example.com
웹사이트: https://test-institute.com
주소: 서울시 강남구 테스트로 123
상세 주소: 2층
```

**확인사항:**
- [ ] 모든 필드 입력 가능
- [ ] 교육 기관 유형 드롭다운 작동
- [ ] "다음 단계" 버튼 클릭 시 검증
- [ ] 필수 필드 누락 시 에러 표시
- [ ] 2단계로 이동

#### 4.3 2단계: 상세 정보 및 약관 동의
**입력 필드:**
- [ ] 교육 기관 소개 (텍스트 영역)
- [ ] 운영 시간
- [ ] 설립 연도
- [ ] 훈련사 수
- [ ] 시설 사진 제공 가능 (체크박스)
- [ ] 교육 기관 인증서 보유 (체크박스)
- [ ] 이용약관 동의 (필수)
- [ ] 개인정보 수집 및 이용 동의 (필수)

**테스트 데이터:**
```
교육 기관 소개: 반려견 행동 교정 및 기본 훈련을 전문으로 하는 교육 기관입니다.
운영 시간: 평일 10:00-19:00, 주말 10:00-17:00
설립 연도: 2018년
훈련사 수: 5
```

**확인사항:**
- [ ] "이전 단계" 버튼으로 1단계로 돌아가기 가능
- [ ] 교육 기관 소개 최소/최대 글자 수 검증
- [ ] 체크박스 선택/해제 작동
- [ ] 약관 동의 체크박스 필수 검증
- [ ] "등록 요청하기" 버튼 클릭
- [ ] 성공 토스트 메시지 표시
- [ ] `/institutes` 페이지로 리다이렉트

#### 4.4 진행 상태 표시
- [ ] 1단계에서 진행바 50% 표시
- [ ] 2단계에서 진행바 100% 표시
- [ ] 현재 단계 강조 표시
