# 훈련사 및 기관 등록 기능 테스트 가이드

## 📋 테스트 개요
TALEZ 플랫폼의 훈련사 및 교육기관 등록 기능을 수동으로 테스트하기 위한 가이드입니다.

## 🔗 테스트 URL
- **훈련사 등록**: `/registration/trainer`
- **기관 등록**: `/registration/institute`  
- **관리자 등록 관리** (승인/거부): `/admin/registrations`

## 1️⃣ 훈련사 등록 테스트

### 접근 방법
1. 브라우저에서 애플리케이션 홈페이지 접속
2. URL 직접 입력: `https://your-domain.replit.dev/registration/trainer`

### 1단계: 개인 정보 입력
필수 입력 항목:
- **이름**: 최소 2글자 (예: 김훈련)
- **이메일**: 유효한 이메일 형식 (예: trainer@test.com)
- **전화번호**: 최소 10글자 (예: 010-1234-5678)
- **주소**: 최소 5글자 (예: 서울시 강남구 테헤란로 123)
- **생년월일**: 날짜 형식 (예: 1990-01-01)

### 2단계: 전문 정보 입력
- **경력**: 드롭다운에서 선택 (예: "5년 이상")
- **전문 분야**: 최소 1개 체크 (예: 기초 복종 훈련, 사회화 훈련)
- **자격증**: 선택 가능 (예: 국가공인 반려동물 행동지도사)
- **자기소개**: 최소 50자 입력
- **서비스 지역**: 입력 (예: 서울시 강남구, 서초구)

### 3단계: 사업자 정보
- **사업자 유형**: 드롭다운 선택 (예: 개인사업자)
- **사업자 등록번호**: 입력 (예: 123-45-67890)
- **상호명**: 입력 (예: 김훈련 반려견 훈련소)
- **시간당 요금**: 숫자 입력 (예: 50000)

### 4단계: 서류 제출
- **프로필 이미지**: 이미지 파일 업로드 (선택)
- **자격증 서류**: PDF 또는 이미지 파일 (다중 선택 가능)
- **포트폴리오 이미지**: 이미지 파일 (다중 선택 가능)

### 최종 제출
- "등록 신청" 버튼 클릭
- 성공 시: 토스트 알림 "훈련사 등록 신청 완료" 표시
- 자동 리다이렉트: `/dashboard/trainer-pending` 페이지로 이동

## 2️⃣ 기관 등록 테스트

### 접근 방법
1. URL 직접 입력: `https://your-domain.replit.dev/registration/institute`

### 1단계: 기본 정보
- **기관명**: 최소 2글자 (예: 테일즈 반려견 아카데미)
- **사업자 등록번호**: 최소 10글자 (예: 123-45-67890)
- **대표자명**: 최소 2글자 (예: 홍길동)
- **설립년도**: 4글자 (예: 2020)
- **이메일**: 유효한 이메일 (예: institute@test.com)
- **전화번호**: 최소 10글자 (예: 02-1234-5678)
- **웹사이트**: 선택 사항

### 2단계: 위치 정보
- **주소**: 최소 10글자 (예: 서울시 강남구 테헤란로 456)
- **상세 주소**: 선택 사항
- **교통편 정보**: 최소 10글자 (예: 지하철 2호선 강남역 3번 출구에서 도보 5분)

### 3단계: 시설 정보
- **전체 면적**: 숫자 입력 (예: 500)
- **강의실 수**: 숫자 입력 (예: 5)
- **최대 수용 인원**: 숫자 입력 (예: 50)
- **시설**: 최소 1개 체크 (예: 실내 훈련장, 야외 훈련장)
- **주차 공간**: 선택 사항
- **특별 시설**: 선택 사항

### 4단계: 서비스 정보
- **서비스 유형**: 최소 1개 체크 (예: 기초 훈련, 문제 행동 교정)
- **대상**: 최소 1개 체크 (예: 퍼피 (6개월 미만), 성견 (1-7년))
- **평일 운영시간**: 입력 (예: 09:00-18:00)
- **주말 운영시간**: 입력 (예: 10:00-17:00)
- **기관 소개**: 최소 100자 입력

### 5단계: 직원 정보
- **총 직원 수**: 숫자 입력 (예: 10)
- **자격증 보유 훈련사 수**: 숫자 입력 (예: 8)
- **수의사 수**: 선택 사항
- **기타 직원**: 선택 사항

### 6단계: 서류 제출
- **사업자 등록증**: PDF 또는 이미지 파일
- **시설 사진**: 이미지 파일 (다중 선택 가능)
- **기관 인증서류**: PDF 또는 이미지 파일 (다중 선택 가능)

### 최종 제출
- "등록 신청" 버튼 클릭
- 성공 시: 토스트 알림 "기관 등록 신청 완료" 표시

## 3️⃣ 관리자 승인 테스트

### 접근 방법 (관리자 계정 필요)
1. 관리자 계정으로 로그인
2. URL: `https://your-domain.replit.dev/admin/registrations`

### 기능
- **등록 신청 목록 조회**: 모든 훈련사/기관 등록 신청 확인
- **상세 정보 확인**: 각 신청서의 세부 내용 검토
- **승인/거부 처리**: 
  - "승인" 버튼: 등록 신청 승인
  - "거부" 버튼: 등록 신청 거부 (사유 입력)

## 🐛 알려진 이슈

### WebSocket 경고 (비치명적)
브라우저 콘솔에 다음 경고가 표시될 수 있습니다:
```
Failed to construct 'WebSocket': The URL 'wss://localhost:undefined/?token=...' is invalid.
```
이는 개발 환경의 WebSocket 설정 관련 경고이며, 등록 기능 자체에는 영향을 주지 않습니다.
