## 📱 **1. 카카오 로그인**

### 공식 가이드
- 공식 문서: https://developers.kakao.com/docs/latest/ko/kakaologin/design-guide

### 주요 정책
**버튼 구성**
- 말풍선 모양의 심볼(Symbol), 레이블(Label), 컨테이너(Container)로 구성

**색상 규정**
- 카카오 고유 색상을 지켜야 하며, 색상 규정에 벗어난 색상 적용 금지
- 타사 로그인 버튼을 강조하기 위해 카카오 버튼의 색상을 지정하지 않는 것은 금지

**레이블**
- 폰트는 30Pt 크기에 OS별 기본 시스템 서체 사용
- 완성형("카카오 로그인")과 축약형("로그인") 사용 가능

**심볼(로고)**
- 카카오를 상징하는 말풍선 모양의 심볼을 기본으로 사용하며, 형태, 비율, 색상 변경 불가
- 심볼 없이 카카오 로그인 버튼 구성 불가

**컨테이너**
- 박스의 radius는 12 픽셀로 적용

**커스터마이징**
- 서비스의 UI/UX에 맞춰 버튼 수정이 필요할 경우, 카카오 고유의 이미지를 해치지 않는 범위 내에서 유동적으로 재구성 가능

## 🟢 **2. 네이버 로그인**

### 주요 정책
**색상**
- Naver Green: HEX #03C75A, RGB 3/199/90

**메시지/레이블**
- 네이버 아이디로 로그인 목적에 부합한다면 한글, 영문 상관없이 변경 가능

**버튼 디자인**
- 네이버 고유 이미지를 해치지 않는 범위 내에서 유연하게 디자인 변경 가능

**로고 사용**
- 검은색 배경에는 흰색 로고, 흰색 배경에는 검은색 로고만 사용 가능
- 로고 수정 금지

**배경색 지침**
- 흰색(#FFFFFF): 로그인 버튼이 위치한 배경과 대비가 확실할 경우 사용
- 흰색, 회색 기본 배경에서는 단독 흰색 사용 불가, 외곽선 포함 버전 사용
- 검은색(#000000): 배경과 대비가 확실할 경우 사용 가능하나, 검은색/어두운 배경에서는 사용 불가

## 🔵 **3. 구글 로그인**

### 공식 가이드
- 공식 브랜딩 가이드라인: https://developers.google.com/identity/branding-guidelines

### 주요 정책
**버튼 동등성**
- Google 로그인 버튼은 다른 타사 로그인 옵션과 적어도 동등한 수준으로 눈에 띄어야 하며, 버튼 크기는 거의 같아야 하고 시각적으로 비슷한 비중을 가져야 함

**색상**
- 눌리지 않은 밝은색 버튼은 #FFFFFF(흰색), 어두운색(파란색) 버튼은 #4285F4
- 버튼 배경은 파란색이나 흰색을 사용할 수 있지만, 항상 흰색 타일에 표시된 표준 색상의 Google 'G' 로고를 포함해야 함

**폰트**
- 버튼 글꼴은 TrueType 글꼴인 Roboto Medium

**로고 규정**
- Google 'G' 로고의 크기나 색상은 변경 불가
- 로고는 표준 색상 버전이어야 하며 흰색 배경에 표시

**간격 및 여백**
- 텍스트 왼쪽 및 오른쪽의 여백은 8dp, 로고는 18dp, 로고와 버튼 텍스트 사이는 24dp

**가로세로 비율**
- 버튼 크기를 조절할 수 있지만, Google 로고가 늘어나지 않도록 가로세로 비율은 유지해야 함

**텍스트 가이드**
- 'Google 계정으로 로그인' 또는 'Google 계정으로 가입' 사용 권장
- 명확한 안내 텍스트가 가까이 있다면 버튼 안에 'Google'이라는 단어만 넣을 수 있음

## 🎯 **공통 주의사항**

1. **표준 디자인 우선 사용** - 세 플랫폼 모두 공식 제공 디자인 사용 권장
2. **브랜드 이미지 훼손 금지** - 로고/심볼의 형태, 비율, 색상 변경 불가
3. **동등한 시각적 비중** - 타사 로그인 버튼과 동등하게 노출
4. **가이드 준수 필수** - 규정을 어길 경우 사후 제재 가능
