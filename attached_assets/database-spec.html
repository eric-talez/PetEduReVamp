TALEZ 데이터베이스 명세서

TALEZ 플랫폼의 데이터베이스 구조 및 스키마를 정리한 문서입니다.

1. 사용자 관련 테이블

users (사용자)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 사용자 고유 ID
username varchar(255) UNIQUE, NOT NULL 사용자 로그인 아이디
email varchar(255) UNIQUE, NOT NULL 사용자 이메일
password varchar(255) NOT NULL 암호화된 비밀번호
name varchar(100) 사용자 실명
profileImage varchar(255) 프로필 이미지 URL
role enum NOT NULL 역할 (ADMIN, USER, TRAINER, INSTITUTION)
phoneNumber varchar(20) 연락처
isActive boolean NOT NULL, DEFAULT true 계정 활성화 상태
verifiedAt timestamp 계정 인증 일시
lastLogin timestamp 마지막 로그인 일시
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
trainers (훈련사)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 훈련사 고유 ID
FK userId integer NOT NULL, REFERENCES users(id) 사용자 ID
certificationCode varchar(100) UNIQUE 인증 코드
certificationType varchar(100) 인증 유형
isVerified boolean NOT NULL, DEFAULT false 인증 여부
institutionId integer REFERENCES institutions(id) 소속 기관 ID
specialties json 전문 분야 (배열)
rating decimal(3,2) DEFAULT 0.0 평점 (0.0-5.0)
reviewCount integer DEFAULT 0 리뷰 수
availableForHire boolean NOT NULL, DEFAULT true 고용 가능 여부
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
trainer_profiles (훈련사 프로필)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 프로필 고유 ID
FK trainerId integer NOT NULL, REFERENCES trainers(id) 훈련사 ID
FK userId integer NOT NULL, REFERENCES users(id) 사용자 ID
bio text NOT NULL 약력
yearsOfExperience integer NOT NULL, DEFAULT 0 경력 연수
certification text 자격증 및 교육이수
introduction text NOT NULL 자기 소개
qualifications text 추가 자격 및 업적
hourlyRate integer NOT NULL, DEFAULT 30000 시간당 요금
acceptsNewClients boolean NOT NULL, DEFAULT true 신규 고객 수락 여부
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
owners (견주)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 견주 고유 ID
FK userId integer NOT NULL, REFERENCES users(id), UNIQUE 사용자 ID
address text 주소
preferences json 선호 설정
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
institutions (기관)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 기관 고유 ID
FK adminUserId integer NOT NULL, REFERENCES users(id) 관리자 사용자 ID
name varchar(255) NOT NULL 기관명
description text 기관 설명
logoUrl varchar(255) 로고 이미지 URL
address text 주소
websiteUrl varchar(255) 웹사이트 URL
contactEmail varchar(255) 연락처 이메일
contactPhone varchar(20) 연락처 전화번호
businessNumber varchar(30) 사업자 등록번호
isVerified boolean DEFAULT false 인증 여부
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시

2. 반려견 관련 테이블

dogs (반려견)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 반려견 고유 ID
name varchar(100) NOT NULL 이름
breed varchar(100) NOT NULL 견종
age varchar(20) 나이
gender enum 성별 (male, female)
weight varchar(20) 체중
neutered boolean DEFAULT false 중성화 여부
FK ownerId integer NOT NULL, REFERENCES users(id) 견주 ID
profileImage varchar(255) 프로필 이미지 URL
specialNeeds text 특별 관리 사항
medicalHistory text 건강 이력
behavior text 행동 특성
createdAt timestamp DEFAULT NOW() 생성 일시
updatedAt timestamp DEFAULT NOW() 수정 일시

3. 훈련 관련 테이블

notebooks (알림장)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 알림장 고유 ID
title varchar(255) NOT NULL 제목
content text NOT NULL 내용
FK trainerId integer NOT NULL, REFERENCES users(id) 훈련사 ID
FK ownerId integer NOT NULL, REFERENCES users(id) 견주 ID
FK dogId integer NOT NULL, REFERENCES dogs(id) 반려견 ID
trainingType varchar(100) NOT NULL 훈련 유형
trainingDate timestamp NOT NULL 훈련 일시
trainingDuration integer NOT NULL 훈련 시간 (분)
trainingGoals text 훈련 목표
trainingProgress integer DEFAULT 0 훈련 진행도 (0-100%)
feedbackType enum DEFAULT 'neutral' 피드백 유형
homework text 과제
isPublic boolean DEFAULT false 공개 여부
tags json 태그 (문자열 배열)
status enum DEFAULT 'draft' 상태
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
training_classes (훈련 수업)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 수업 고유 ID
title varchar(255) NOT NULL 수업명
description text 수업 설명
FK trainerId integer NOT NULL, REFERENCES trainers(id) 훈련사 ID
FK institutionId integer REFERENCES institutions(id) 기관 ID
classType varchar(50) NOT NULL 수업 유형 (개인/그룹)
category varchar(100) 수업 카테고리
level varchar(50) 난이도
price integer NOT NULL 가격
capacity integer 최대 수용 인원
duration integer NOT NULL 기간 (일)
sessions integer NOT NULL 세션 수
imageUrl varchar(255) 이미지 URL
startDate timestamp 시작 일시
endDate timestamp 종료 일시
isActive boolean DEFAULT true 활성화 여부
isPublic boolean DEFAULT true 공개 여부
requirements text 참여 요구사항
status varchar(50) DEFAULT 'upcoming' 상태
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시

4. 교육 콘텐츠 관련 테이블

curriculum_sections (커리큘럼 섹션)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 섹션 고유 ID
FK curriculumId integer NOT NULL, REFERENCES curriculums(id) 커리큘럼 ID
title varchar(255) NOT NULL 섹션명
description text 섹션 설명
order integer NOT NULL 정렬 순서
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
curriculums (커리큘럼)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 커리큘럼 고유 ID
title varchar(255) NOT NULL 커리큘럼명
description text 커리큘럼 설명
FK creatorId integer NOT NULL, REFERENCES users(id) 제작자 ID
category varchar(100) 카테고리
level varchar(50) 난이도
thumbnailUrl varchar(255) 썸네일 이미지 URL
isPublished boolean DEFAULT false 출판 여부
isFeatured boolean DEFAULT false 추천 여부
tags json 태그 (문자열 배열)
price integer DEFAULT 0 가격
discountPrice integer 할인 가격
isSalable boolean DEFAULT false 판매 가능 여부
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
training_videos (훈련 영상)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 영상 고유 ID
title varchar(255) NOT NULL 영상 제목
description text 영상 설명
videoUrl varchar(255) NOT NULL 영상 URL
thumbnailUrl varchar(255) 썸네일 이미지 URL
durationSeconds integer 재생 시간 (초)
FK uploaderId integer NOT NULL, REFERENCES users(id) 업로더 ID
category varchar(100) 카테고리
level varchar(50) 난이도
isFree boolean DEFAULT false 무료 여부
isPublished boolean DEFAULT false 출판 여부
viewCount integer DEFAULT 0 조회수
likeCount integer DEFAULT 0 좋아요 수
dislikeCount integer DEFAULT 0 싫어요 수
tags json 태그 (문자열 배열)
FK sectionId integer REFERENCES curriculum_sections(id) 섹션 ID
order integer 정렬 순서
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시

5. 인증 및 배지 관련 테이블

certificates (수료증)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 수료증 고유 ID
certificateNumber varchar(100) NOT NULL, UNIQUE 인증 번호
FK userId integer NOT NULL, REFERENCES users(id) 사용자 ID
FK courseId integer NOT NULL, REFERENCES courses(id) 과정 ID
issueDate timestamp NOT NULL 발급일
expiryDate timestamp 만료일
templateId varchar(100) 템플릿 ID
pdfUrl varchar(255) PDF URL
isVerified boolean DEFAULT true 인증 여부
verificationCode varchar(100) NOT NULL, UNIQUE 인증 코드
metadata json 추가 메타데이터
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시
badges (배지)
컬럼명 데이터 타입 제약조건 설명
PK id serial PRIMARY KEY 배지 고유 ID
name varchar(100) NOT NULL 배지명
description text NOT NULL 배지 설명
imageUrl varchar(255) NOT NULL 배지 이미지 URL
category varchar(50) NOT NULL 배지 카테고리
level integer DEFAULT 1 배지 레벨
pointValue integer DEFAULT 0 포인트 가치
requirement text 획득 조건
isHidden boolean DEFAULT false 숨김 여부
isActive boolean DEFAULT true 활성화 여부
createdAt timestamp NOT NULL, DEFAULT NOW() 생성 일시
updatedAt timestamp NOT NULL, DEFAULT NOW() 수정 일시

참고: 이 데이터베이스 명세서는 현재 시스템 상태를 기준으로 작성되었으며, 향후 업데이트에 따라 변경될 수 있습니다.
