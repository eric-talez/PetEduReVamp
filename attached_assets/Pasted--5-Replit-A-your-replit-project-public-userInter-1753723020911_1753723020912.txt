## 🎯 6. 실제 사용 가이드

### A. Replit에서 설정 순서

1. **환경 변수 설정** (Secrets 탭에서):
   ```
   ADMIN_TOKEN=your-secret-admin-token-here
   WEBHOOK_URL=https://hooks.slack.com/your-webhook-url (선택사항)
   ```

2. **package.json에 의존성 추가**:
   ```json
   {
     "dependencies": {
       "@replit/database": "^2.0.1",
       "express": "^4.18.0"
     }
   }
   ```

3. **파일 업로드**:
   - 위의 모든 JavaScript 파일들을 public 폴더에 업로드
   - server.js에 API 엔드포인트 코드 추가

### B. 모니터링 시작

1. **Replit 실행 후 접속**:
   - 메인 사이트: `https://your-repl-name.replit.app`
   - 대시보드: `https://your-repl-name.replit.app/dashboard`
   - 관리자 대시보드: `https://your-repl-name.replit.app/admin/dashboard?token=your-admin-token`

2. **실시간 모니터링 확인**:
   - 브라우저 개발자 도구(F12) → Console 탭에서 로그 확인
   - 대시보드에서 실시간 데이터 확인

### C. 문제 발생시 대응

1. **버튼 무반응 감지시**:
   - 자동으로 사용자에게 알림 표시
   - 관리자 대시보드에 즉시 리포트
   - 콘솔에 상세 정보 로깅

2. **페이지 연결 실패시**:
   - 깨진 링크 시각적 표시
   - 사용자에게 대안 안내
   - 자동으로 재검사 수행
