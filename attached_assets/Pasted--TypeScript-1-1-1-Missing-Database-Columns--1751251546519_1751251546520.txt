## 9. 일괄 실행 스크립트

다음 명령어들을 순서대로 실행하세요:

```bash
# 1. 데이터베이스 마이그레이션 (스키마 업데이트)
npm run db:migrate

# 2. TypeScript 타입 체크
npm run type-check

# 3. 린트 및 자동 수정
npm run lint:fix

# 4. 빌드 테스트
npm run build
```
