TALEZ API 명세서

TALEZ 플랫폼에서 사용되는 API 목록과 상세 정보를 정리한 문서입니다.

1. 사용자 인증 API

POST

회원가입

/api/register

새로운 사용자 계정을 생성합니다.

요청 파라미터

필드명 타입 필수 설명
username string Y 사용자 아이디
email string Y 사용자 이메일
password string Y 비밀번호
name string N 사용자 실명
phoneNumber string N 연락처

응답

{
  "id": 123,
  "username": "user123",
  "email": "user@example.com",
  "name": "홍길동",
  "role": "USER"
}
POST

로그인

사용자 인증 및 세션 생성

요청 파라미터

필드명 타입 필수 설명
username string Y 사용자 아이디
password string Y 비밀번호

응답

{
  "id": 123,
  "username": "user123",
  "email": "user@example.com",
  "name": "홍길동",
  "role": "USER"
}
