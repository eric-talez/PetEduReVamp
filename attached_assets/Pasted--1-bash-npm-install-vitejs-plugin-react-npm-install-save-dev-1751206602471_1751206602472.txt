## 실행 순서

1. **의존성 설치**
2. **vite.config.ts 수정**
3. **shared/types.ts 생성**
4. **tsconfig.json 수정**
5. **빌드 실행**: `npm run build`

이 순서대로 진행하면 대부분의 빌드 오류가 해결됩니다!
