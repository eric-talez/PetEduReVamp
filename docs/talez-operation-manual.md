## 2. 권한 구조

### 2.1 사용자 권한 레벨
1. 비회원 (user)
   - 접근 가능: 메인 메뉴, 쇼핑
   - 제한 사항: 강의 등록, 메시지 전송 불가

2. 반려견 주인 (pet-owner)
   - 접근 가능: 메인 메뉴 + 나의 학습 메뉴
   - 특별 권한: 반려견 프로필 관리, 강의 수강

3. 훈련사 (trainer)
   - 접근 가능: 메인 메뉴 + 훈련사 메뉴
   - 특별 권한: 강의 개설, 수강생 관리

4. 기관 관리자 (institute-admin)
   - 접근 가능: 메인 메뉴 + 기관 관리 메뉴
   - 특별 권한: 소속 훈련사/강의 관리

5. 시스템 관리자 (admin)
   - 접근 가능: 전체 메뉴
   - 특별 권한: 시스템 설정, 권한 관리

## 3. 주요 기능 및 조건값

### 3.1 회원 관리
1. 회원가입
   - 필수 정보: 이메일, 비밀번호, 이름
   - 선택 정보: 프로필 이미지, 소개
   - 인증 조건: 이메일 인증 필수
   
2. 본인인증
   - 방법: 휴대폰 인증
   - 유효기간: 1년
   - 재인증 조건: 비밀번호 변경 시

### 3.2 강의 관리
1. 강의 등록 (훈련사)
   - 필수 항목: 제목, 설명, 가격, 기간
   - 최소 강의 시간: 30분
   - 최대 수강 인원: 20명
   - 승인 필요: 기관 관리자 승인

2. 수강 신청 (반려견 주인)
   - 신청 조건: 본인인증 완료
   - 결제 방식: 선결제
   - 취소 정책: 7일 이내 100% 환불

### 3.3 훈련사 관리
1. 훈련사 등록
   - 필수 자격: 공인 자격증
   - 경력 조건: 최소 1년
   - 평가 기준: 4.0/5.0 이상 유지

2. 수수료 정책
   - 기본 수수료: 20%
   - 우수 훈련사: 15%
   - 정산 주기: 월 1회

### 3.4 기관 관리
1. 기관 등록
   - 필수 서류: 사업자등록증
   - 시설 기준: 최소 50평
   - 인증 조건: 현장 실사

2. 기관 등급
   - 일반: 기본 기능
   - 프리미엄: 추가 기능 + 우선 노출
   - 갱신 주기: 1년

### 3.5 결제/정산
1. 결제 설정
   - 최소 결제액: 10,000원
   - 결제 수단: 카드, 계좌이체
   - 할부 옵션: 3개월까지 무이자

2. 정산 정책
   - 정산 주기: 월 1회
   - 수수료 공제: 결제 금액의 15-20%
   - 최소 정산액: 50,000원
