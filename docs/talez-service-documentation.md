
# TALEZ 서비스 상세 기능 명세서

## 1. 사용자 관리 시스템

### 1.1 회원가입/로그인
- 일반 회원가입
  - 이메일 인증
  - 휴대폰 본인인증 (토스 연동)
  - 약관 동의
- 소셜 로그인
  - 카카오
  - 네이버
  - 구글
- 회원 유형
  - 일반 사용자 (user)
  - 반려견 주인 (pet-owner)
  - 훈련사 (trainer)
  - 기관 관리자 (institute-admin)
  - 시스템 관리자 (admin)

### 1.2 프로필 관리
- 기본 정보 관리
  - 프로필 사진
  - 이름
  - 연락처
  - 주소
- 전문가 프로필 (훈련사)
  - 자격증 정보
  - 경력사항
  - 전문 분야
  - 활동 지역
- 반려견 프로필
  - 기본 정보 (이름, 나이, 품종)
  - 건강 기록
  - 훈련 이력
  - 예방접종 기록
